# Age Detection Using Vitiligo

A deep learning-based approach to estimate age using skin images affected by vitiligo.

## Technologies Used
- Python
- OpenCV
- TensorFlow
- CNN

## How to Run
1. Clone this repo
2. Install requirements
3. Run `main.py`
